== Description ==
TCD Google Maps provides a simple shortcode to display the Google Maps.
You can display various styles of GoogleMaps just put a shortcode to posts, pages, or Text Widget.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Input your map data at admin page and you get the shortcode
4. Copy and Paste the shortcode in posts, pages, or Text Widgets

If you need additional help [read our detailed instructions](http://design-plus1.com/tcd-w/2016/01/googlemap.html)
